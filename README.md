## Digital Business Card
This is a digitized version of a typical business card, the idea was to make an easily accessible way to share professional contact information. The design features a self portrait with signifying logos for different ways to contact me (instagram, email, and my portfolio website), my personal logo featured in the top left, and a few descriptive words for me. The project is an assignment for the Spring 2026 Web Typography taught by Lara Henerson at the University of Massachusetts: Dartmouth as a way to practice HTML and CSS code and explore typography for the web.
## Process
A physical sketch is always the starting point for my designs to flesh out the many different design ideas I come up with. This then leads to a high fidelity mock up in figma to organize fonts, logos, and bacground image. During the ideation process learning how to setup HTML and CSS code documents and code functioning sites using CSS grid was essential to creating this functioning website.
## Personal Branding
The branding on this digital business card is something I have been developing for a long time. The smokey purple and light teal have been colors I've repetedly reached for over the course of my life so they have become staples in my personal work. My personal logo is a play on my name, my first name in my signature and the heart around the signature to refrence my last name, Hart. The design creates not only a tag for my physical work but logo for myself.
